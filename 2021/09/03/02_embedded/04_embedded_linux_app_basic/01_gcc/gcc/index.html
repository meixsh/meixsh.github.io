1.3 常用编译选项

常用选项 描述
-E 预处理，开发过程中想快速确定某个宏可以使用-E -dM
-c 把预处理、编译、汇编都做了，但是不链接
-o 指定输出文件
-I 指定头文件目录
-L 指定链接时库文件目录
-l 指定链接哪一个库文件

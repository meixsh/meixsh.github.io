1. 背景知识

欧洲计算机制造商协会 ECMA TC39技术委员会 制定了ECMAScript标准

国际标准化组织 ISO 国际电工委员会 IEC, ISO/IEC-16262 标准收录了ECMAScript

ECMA-262

  • ECMA标准里程碑

    • ES5 2009年
    • ES6/ES2015 2015年
    • ES7/ES2016 2016年
    • ES8/ES2017 2017年
    • ES9/ES2018 2018年
    • ES10/ES2019 2019年

  • W3C （WorldWideWeb Consortium）制定DOM标准

  • HTML5 统一BOM标准

  • JavaScript是一门用来与网页交互的脚本语言，包含3个部分

    • ECMAScript：ECMA-262定义的核心功能
    • 文档对象模型DOM：定义与网页交互的接口
    • 浏览器对象模型BOM：定义与浏览器交互的接口

  • 五大浏览器基本对ES5支持完善

    • IE 微软
    • Firefox 网景
    • Chrome 谷歌
    • Safari 苹果

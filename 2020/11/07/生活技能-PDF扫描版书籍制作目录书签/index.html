技能-PDF扫描版书籍制作目录书签

发表于 2020-11-07 分类于 技能 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

最近利用学校图书馆下了几本专业书籍的电子版，可是这些电子版书籍是扫描版的PDF格式，不支持搜索且无目录书签，十分不利于阅读学习，前者暂时不知道怎么解决同时也不是非常的紧急，但是没有后者目录书签的功能，可真是太令我头疼了。遂上网调研寻求解决方法，利用万能的谷歌，加上关键字搜索，如何为扫描版PDF添加目录，参考摸索得到下面的解决方案，真叫一个爽！

  • 推荐软件环境：Notepad++、FreePic2Pdf、福昕阅读器高级版（支持页面编辑）[非必须]

  • 寻找待制作目录书签电子书的目录文件，这里可以参考京东自营店该书籍详情页的目录信息或者豆瓣提供的书籍目录信息，此时直接复制粘贴至新建的目录文档。但是，可能在这两个地方找不到完整的目录信息，或者压根就没有目录信息，此时请另谋出路，利用谷歌搜索或者自己编辑，自求多福吧。

  • 使用Notepad++打开第一步新建的目录文档，预处理该文档，使之符合后续FreePic2Pdf书签目录制作软件的目录格式规范

    • ctrl + f 进入替换对话框，勾选正则表达式，按下面步骤进行正则查找替换
    • 三级标题插入2个tab键：^(\d{1,2}\.\d{1,2}\.\d{1,2})、\t\t\1
    • 二级标题插入1个tab键：^(\d{1,2}\.\d{1,2})、\t\1
    • 页码数字之前插入tab键：(\d{1,3})$、\t\1

  • 打开FreePic2Pdf，依次选择：更改PDF → 从PDF取书签，设置好待添加目录的PDF文件以及用于存放生成的目录接口文件的路径，点击开始，即可完成目录接口文件的生成。

  • 使用Notepad++打开接口文件FreePic2Pdf_bkmk.txt，将经过预处理的目录文档内容拷贝至该接口文件并保存。

  • 使用Notepad++打开接口文件FreePic2Pdf.itf，修改字段BasePage的值为扫描版PDF文件正文第一页所在的实际页数，即使用PDF阅读器打开该PDF文档，定位到该页，并查看PDF阅读器下方页码区域显示的页码数字。

  • 使用FreePic2Pdf，依次选择：更改PDF → 往PDF挂书签，将配置好的目录接口文件信息写入PDF文档中。切记如果此步骤之前打开了该PDF文档，请在此步骤进行前务必关闭该PDF文档，否则生成接口文件失败。

  • 打开制作好目录书签的文档，享受阅读。

注: 可以先生成目录接口文件，直接在该文件中预处理目录文档信息，可以减少一次文件的新建和拷贝操作。
